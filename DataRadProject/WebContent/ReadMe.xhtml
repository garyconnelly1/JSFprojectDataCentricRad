ReadMe

This is the readme file for the data centric Rad project in y3.

This project is very easy to run. Just run the index.xhtml on the tomcat server.

Once you are presented with the index file, simply click on the link that you want to inspect.

The country link shows you a list of countries grabbed from the geography database.

In it you are able to add, update or delete the country you wish.

Once you are finished with that you can look at the regions link on the homepage.

In that you are able to view all the regions currently in the database, aswell as delete whatever region you want

Then you have cities. In there cou can view the list of cities database, and you can also add your own city. However I could not the the full details to work, but you can see my attempts at it in the java code.

You can also view the heads of state database. This database is displayed by connecting to a mongo database which I created myself.

There is also an attemp at getting the find cities to work, but as it doesnt work I decided not to provide a link to non working code on the web application. However, if you wish to view the attempts at it, you can see it in my java code.
